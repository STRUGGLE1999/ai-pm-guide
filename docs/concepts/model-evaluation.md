# 模型评估与质量

AI 产品上线前后都需要持续评估，质量管理是 PM 的核心工作之一。

## 知识点清单

| 知识点 | PM 要掌握的重点 | 面试可能怎么问 |
| --- | --- | --- |
| 评估集 | 覆盖高频、困难、边界和风险样例。 | 评估集 对 AI 产品设计有什么影响？ |
| 人工评分 | 定义准确性、完整性、格式、引用和可执行性标准。 | 人工评分 对 AI 产品设计有什么影响？ |
| 线上反馈 | 将用户采纳、编辑、重试和差评沉淀为优化线索。 | 线上反馈 对 AI 产品设计有什么影响？ |
| A/B 测试 | 比较不同模型、Prompt、检索策略和交互流程。 | A/B 测试 对 AI 产品设计有什么影响？ |
| 回归测试 | 每次 Prompt 或模型变更后检查核心样例。 | 回归测试 对 AI 产品设计有什么影响？ |
| 幻觉识别 | 关注无依据回答、错误引用和过度自信表达。 | 幻觉识别 对 AI 产品设计有什么影响？ |
| 引用质量 | 检查检索片段是否支持最终答案。 | 引用质量 对 AI 产品设计有什么影响？ |
| 延迟体验 | 评估首字时间、总耗时和等待反馈。 | 延迟体验 对 AI 产品设计有什么影响？ |
| 稳定性 | 批量任务中看输出结构、结果波动和失败率。 | 稳定性 对 AI 产品设计有什么影响？ |
| 成本监控 | 按任务类型追踪 token、模型和重试成本。 | 成本监控 对 AI 产品设计有什么影响？ |
| 安全评估 | 覆盖敏感内容、越权访问和提示词注入。 | 安全评估 对 AI 产品设计有什么影响？ |
| 质量看板 | 让业务、产品和研发看到同一套质量信号。 | 质量看板 对 AI 产品设计有什么影响？ |

## 学习方法

1. 先用一句话解释概念，避免陷入纯技术细节。
2. 找一个真实产品场景，说明它如何改变用户流程。
3. 写出风险、指标和验收标准。

## 练习任务

选择 3 个知识点，各写一个「用户场景 + 产品方案 + 指标」的小卡片。

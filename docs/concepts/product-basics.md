# 产品经理基本功

AI 产品仍然先是产品，需要需求、用户、流程、指标和协作能力。

## 知识点清单

| 知识点 | PM 要掌握的重点 | 面试可能怎么问 |
| --- | --- | --- |
| 需求分析 | 从用户任务出发，而不是从“接入大模型”出发。 | 需求分析 对 AI 产品设计有什么影响？ |
| 用户研究 | 围绕最近一次真实任务追问输入、输出、判断标准和错误成本。 | 用户研究 对 AI 产品设计有什么影响？ |
| 竞品分析 | 用任务链路比较产品，不只做功能表。 | 竞品分析 对 AI 产品设计有什么影响？ |
| PRD | 额外写清模型行为、输入输出、评估标准和异常兜底。 | PRD 对 AI 产品设计有什么影响？ |
| 指标体系 | 覆盖使用、质量、效率、成本和风险五层指标。 | 指标体系 对 AI 产品设计有什么影响？ |
| 数据分析 | 通过触发率、采纳率、重试率、编辑率定位问题。 | 数据分析 对 AI 产品设计有什么影响？ |
| B 端流程 | 关注权限、安全、集成、审计、交付和运营责任。 | B 端流程 对 AI 产品设计有什么影响？ |
| ROI 测算 | 把 AI 能力转化为降本、提效、增收或风险降低。 | ROI 测算 对 AI 产品设计有什么影响？ |
| 灰度发布 | 降低模型不稳定和流程误操作带来的上线风险。 | 灰度发布 对 AI 产品设计有什么影响？ |
| 需求优先级 | 优先选高频、耗时、数据可得、错误成本可控的场景。 | 需求优先级 对 AI 产品设计有什么影响？ |
| 验收标准 | 用样例集和可观察指标替代“生成得不错”。 | 验收标准 对 AI 产品设计有什么影响？ |
| 跨团队协作 | 连接业务、研发、算法、法务、安全和运营。 | 跨团队协作 对 AI 产品设计有什么影响？ |

## 学习方法

1. 先用一句话解释概念，避免陷入纯技术细节。
2. 找一个真实产品场景，说明它如何改变用户流程。
3. 写出风险、指标和验收标准。

## 练习任务

选择 3 个知识点，各写一个「用户场景 + 产品方案 + 指标」的小卡片。

# 面经来源与整理规则

## 来源范围

- [牛客网](https://www.nowcoder.com/)：AI 产品经理/大模型产品经理面经、产品经理面试讨论
- [力扣讨论区](https://leetcode.cn/discuss/)：产品、AI 应用和技术产品相关面试讨论
- [小红书](https://www.xiaohongshu.com/)：AI 产品经理转型经验、面试复盘、学习路线分享
- [知乎](https://www.zhihu.com/)：AI 产品经理岗位、能力模型和面试经验问答
- [掘金](https://juejin.cn/)：AI 应用产品、Agent、RAG、Prompt 工程实践文章
- [人人都是产品经理](https://www.woshipm.com/)：AI 产品案例、产品设计和行业观察

## 整理规则

1. 不复制原帖正文。
2. 只提炼题型、考点、能力要求和准备建议。
3. 面试题统一改写为原创题干和回答框架。
4. 保留平台级来源链接，便于读者自行检索。
5. 对无法核验的个案经验，不作为事实结论。

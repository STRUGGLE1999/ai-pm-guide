# 实战案例

- [AI 知识库](/practice/ai-knowledge-base)：让员工基于企业资料进行可信问答、摘要和知识复用。
- [智能客服 Bot](/practice/customer-service-bot)：把 FAQ、追问、转人工和工单摘要串成闭环。
- [AI 搜索](/practice/ai-search)：把搜索升级为答案、引用来源和下一步推荐。
- [内容生成工具](/practice/content-generation)：用模板、素材、审核和版本管理提升内容生产效率。
- [Agent 工作流](/practice/agent-workflow)：让 AI 规划步骤、调用工具并在人类确认下执行任务。
- [智能报表助手](/practice/smart-report)：从指标查询、异常识别到复盘报告生成。
- [销售线索处理 Agent](/practice/sales-agent)：补全线索、生成客户画像和跟进话术。
- [会议纪要助手](/practice/meeting-notes)：从转写到摘要、行动项和跟进提醒。
- [合同审查助手](/practice/contract-review)：识别条款风险、给出依据和修改建议。
- [数据分析 Copilot](/practice/data-analyst-copilot)：帮助业务同学自然语言查数和解释波动。
- [AI 简历优化助手](/practice/resume-coach)：围绕岗位要求优化项目表达和 STAR 结构。
- [行业研究 Agent](/practice/research-agent)：自动搜集资料、归纳观点并输出研究简报。
- [企业培训助手](/practice/training-assistant)：把课程资料转成测验、学习路径和答疑助手。
- [运营 Copilot](/practice/ops-copilot)：支持活动文案、用户分群和复盘分析。
- [用户反馈挖掘](/practice/product-feedback-mining)：从评论、工单和访谈中提取高频问题和需求机会。
